# Презентация решения
## Архитектура решения
Программа состоит из методов, часть из которых было написано ранее при прохождении курса, часть - во время выполнения данного задания. А также из тела программы, содержащей выводы информации в консоль и вызовы методов

## Краткое описание алгоритма
Для работы нам потребуется скачать и установить следующее программное обеспечеине:

1. Зададим размерность массива с клавиатуры, попутно проверив ввод на корректность
2. Создадим и заполним исходный массив с клавиатуры
3. Через цикл посчитаем количество элементов исходного массива с длинной элемента меньшим или равным трем 

> Данное действие необходимо для понимания размера результирующего массива

4. Далее через цикл запишем элементы исходного массива с длинной элемента меньшим или равным трем в результирующий массив
5. Выведем исходный и результирующие массивы в консоль
